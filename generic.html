Propuesta

El proyecto propone el uso de energía amigable con el medio ambiente para alimentar la iluminación y los equipos multimedia propuestos. Este objetivo se cumplirá mediante la instalación de Tapetes generadores de Energía, estos consisten un Sistema Piezoeléctrico, que, aprovechando el movimiento creado por la afluencia de usuarios, generará y almacenará energía mecánica. La instalación será dedicada una exposición sobre los destinos turísticos más importantes de cada estado de nuestro país, así como sus leyendas y elementos más representativos; todo esto acompañado de una App y una página web.

De este modo, México en unos pasos es también una aplicación de Realidad Aumentada para dispositivos móviles. Para el uso de la aplicación, el usuario accede a la cámara del dispositivo, escanea los marcadores para mostrar el contenido en Realidad Aumentada. Por su versatilidad, el proyecto puede ser adaptado a diferentes locaciones de gran afluencia, no sólo en la presente propuesta.
